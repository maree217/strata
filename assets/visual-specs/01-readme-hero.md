# Visual Spec: Hero Banner

**Location:** Main README.md (top of page)
**Type:** Hero Banner / Header Image
**Dimensions:** 1200x630px (social preview compatible)

---

## Purpose

First impression. Communicates the core value proposition in 3 seconds.

---

## Content Elements

### Headline
```
Strata
```

### Tagline
```
A framework for building AI correctly
and managing it continuously.
```

### Key Visual Elements

1. **Three stacked layers** (representing the architecture)
   - Layer 1 (bottom): Infrastructure - solid, foundational
   - Layer 2 (middle): Intelligence - glowing, neural
   - Layer 3 (top): Experience - clean, user-facing

2. **Continuous loop arrow** (representing portfolio intelligence)
   - Wrapping around or connecting to the layers
   - Suggests ongoing iteration

3. **Metrics/data points** (subtle background elements)
   - Small charts, percentages
   - Suggests visibility and tracking
