## Screenshot Guidelines

### Technical Specs
- Resolution: 2x/Retina (2400x1600 for 1200x800 display)
- Format: PNG
- Browser: Clean Chrome/Safari frame
- No personal data visible

### Content Guidelines
- Use realistic fake data
- Generic company names
- Common first name + last initial
- Realistic number ranges

### Visual Style
- Light mode as default
- Clean, no extra toolbars
- Minimal browser chrome
- Consistent viewport size

---

## Variations Needed

1. **Individual screens** - All 6 dashboard pages
2. **Collage** - Overview of all screens
3. **Animated GIF** - Clicking through pages
4. **Mobile views** - If responsive
5. **Dark mode** - All screens
6. **Feature highlights** - With callout annotations
