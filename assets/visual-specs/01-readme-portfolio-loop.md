# Visual Spec: Portfolio Intelligence Loop

**Location:** README.md, philosophy/portfolio-intelligence.md
**Type:** Cycle Diagram / Flowchart
**Dimensions:** 800x500px

---

## Purpose

Show that AI is continuous, not a project with an end date. Visualize the never-ending loop of manage → monitor → iterate.

---

## Content Elements

### The Continuous Loop

```
                    IDEA
                      │
                      ▼
              ┌───────────────┐
              │   EXPERIMENT  │
              └───────┬───────┘
                      │
                      ▼
              ┌───────────────┐
              │    DEPLOY     │
              └───────┬───────┘
                      │
          ┌───────────┴───────────┐
          │                       │
          ▼                       │
    ┌───────────┐                 │
    │  MONITOR  │◀────────────────┤
    └─────┬─────┘                 │
          │                       │
          ▼                       │
    ┌───────────┐                 │
    │  ITERATE  │─────────────────┘
    └───────────┘
          │
          │ (loop continues forever)
          ▼
```

### Key Message

```
"AI is not a project you finish.
 It's a capability you operate."
```

### Stage Details

| Stage | Description | Duration |
|-------|-------------|----------|
| **Idea** | Capture opportunity | Days |
| **Experiment** | Test feasibility | Weeks |
| **Deploy** | Launch to users | Days |
| **Monitor** | Track performance | Ongoing |
| **Iterate** | Improve based on data | Weekly/Monthly |

## Visual Concept: Circular Loop

```
                 IDEA
                  │
            ┌─────┴─────┐
            │           │
            ▼           │
       EXPERIMENT       │
            │           │
            ▼           │
         DEPLOY         │
            │           │
            ▼           │
        MONITOR ◀───────┤
            │           │
            ▼           │
        ITERATE ────────┘
            │
            │ (continues)
            ▼
```

### Alternative: Infinity Loop

```
        ┌─────────────────────────────┐
        │                             │
    MONITOR ──▶ ITERATE ──▶ MONITOR ──┘
        │
        ▼
    (never ends)
```

## Comparison: Traditional vs Portfolio

### Traditional Project Thinking
```
Start ──────────────────▶ DONE ✓
          (ends)
```

### Portfolio Thinking
```
Start ──▶ Deploy ──▶ Monitor ──▶ Iterate ──▶ Monitor ──▶ ...
                        (never ends)
```
