## Screenshot Guidelines

### Browser Frame
- Use clean browser chrome (Safari or Chrome)
- No personal info visible
- Light mode preferred for screenshots

### Data
- Use realistic but fake data
- Company names: Generic (Acme Corp, TechCo)
- Names: Common names (J. Smith, M. Chen)
- Numbers: Realistic ranges

### Quality
- Retina/2x resolution
- PNG format
- No compression artifacts
- Consistent sizing
